## Software & Game Developer
My primary focus is on developing functional mobile utilities and narrative-driven games. Currently, I am designing the architecture for SakayKo, a micro-mobility sharing system for E-bike and scooter rentals, managing the project structure from initial ERDs to comprehensive Gantt charts.

I am highly engaged in technical problem-solving, having recently prepared for the ITECH Connect programming competition. In my collaborative work, I lead teams of student developers to build out robust project requirement documents (PRDs) and organize academic events, balancing technical execution with creative design.
